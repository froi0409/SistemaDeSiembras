Para que la aplicacion de MySQL se conecte con la de JavaNetBeans se deben de seguir los siguientes pasos:
	1. Ejecutar el archivo User_moon_creation.sql para registrar al usuario que la app usa para las conexiones.
	2. Ejecutar el archivo SCHEMA_SOWING_MOON_2.sql para crear la db y tablas a usar.
	3. Ejecutar el archivo INSERT INICIALEs.sql para tener los registros de FASE y DIA necesarios para el funcionamiento de la aplicacion.
	
	
NOTA: El archivo SCHEMA_SOWING_MOON.sql era el archivo inicial de la DB, pero se realizaron varios cambios a la DB.
NOTA 2: El archivo INSERT DE PRUEBA.sql tiene como insercion varios usuarios y algunas actividades en AGENDA que se usaron para prueba de obtencion de informacion.
